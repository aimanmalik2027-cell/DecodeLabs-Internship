# Project 4 – Social Media Analytics

## Brand
GlowNest Skincare

## Project Overview

This project was completed as part of my Digital Marketing Internship at DecodeLabs.

The purpose of this project was to analyze social media performance using key metrics such as reach, engagement, likes, comments, and shares. The analysis helped identify high-performing content and provided recommendations for improving future marketing strategies.

## Objectives

- Track Social Media KPIs
- Measure Audience Engagement
- Analyze Content Performance
- Interpret Social Media Data
- Improve Marketing Strategy

## Key Performance Indicators (KPIs)

- Reach
- Impressions
- Likes
- Comments
- Shares
- Saves
- Engagement Rate
- Profile Visits
- Follower Growth

## Performance Insights

- Customer testimonial content achieved the highest engagement.
- Educational skincare posts generated strong reach.
- Reels performed better than static image posts.
- Consistent posting improved audience interaction.

## Recommendations

- Publish more educational content.
- Increase the number of Instagram Reels.
- Use trending hashtags.
- Maintain a consistent posting schedule.
- Add strong Call-to-Action (CTA) in every post.
